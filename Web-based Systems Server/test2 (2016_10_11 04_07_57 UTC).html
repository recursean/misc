HTML 5

  • Successor to both HTML 4 and XHTML 1
  • Immediate goals (see the W3C's W3C Relaunches HTML Activity and Architectural vision behind the HTML/XHTML2/Forms Chartering):
    • Add new language features so that web page authors will want to adopt the new language and web browser developers will therefore want to support it
    • Support both HTML and XML as different serializations of a single underlying language
    • Design the language and serializations so that it is relatively easy to convert automatically between the serializations, e.g., so that a document that works well in a browser can be readily converted for processing by XML software
  • Longer-term goal
    • Increase percentage of browser pages that are either written in XML or easily transformable into XML
